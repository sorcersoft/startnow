Managing "My Groups"

The "My Groups" menu contents can be managed using the "Configure..." menu item that is in that menu.  Selecting the "Configure..." menu item will bring up a dialog as shown below.

This dialog will allow empty groups to be deleted from the list.  An empty group

will be shown in the list using the string "<empty>" next to the item, instead of

showing the number of services/tools that are currently active in that group.

 

Select the empty group name in the list, and then select the "Delete" button to remove that group name from view.
