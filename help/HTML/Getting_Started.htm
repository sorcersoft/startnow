Getting Started

The EOI Desktop environment provides access to a dynamic set of services that can be added and removed by your administrators.  When you first start using the EOI Desktop environment, you will need to select the groups of services or tools that you want to have access to.  Tools are grouped together by your administration to help you see new tools that are particular to your needs, as well as to limit the number of tools you need to sort through.

 

The image above shows the "My Groups" menu that shows the list of groups that are visible in the system and lets you select the ones you want to have active in your environment.  Also visible in that image are the tabs, "My Apps", "More Apps", and "All Apps".  The "My Apps" tab shows only the desktop.  Items in the trees displayed in the "More Apps" and "All Apps" tabs can be dragged to and dropped on the desktop for quicker access as shown in the image below.

 

The "My Groups" menu contains a menu item to manage the visible group names, and the list of active groups.  The image below shows a sample "My Groups" menu.

The checked group names will be visible in the "More Apps" tree as shown in the image below.

The applications that are in each group will be visible when the respective group folder is opened by selecting it.  The "All Apps" tab tree will always show all of the selected groups so that you can easily browse the other groups to find tools that are useful to you.  If there are disparate groups of tools that you are interested in, you can drag the respective icons from the trees onto the desktop as described above and access them that way, instead of through the trees.

Finally. the "My Apps" labeled dropdown list provides instance access to all applications that are in your selected groups.  This list can be quickly navigated to select tools as well.
